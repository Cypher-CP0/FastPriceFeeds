## Installation 
To set up FastPriceFeeds locally, follow these steps: 
1.	Clone the repository:
```git clone https://github.com/Cypher-CP0/FastPriceFeeds.git```
2.	Navigate to the project directory:
```cd FastPriceFeeds```
3.	Install dependencies:
```npm install```
4.	Run
```npm run dev```
